Developer and musician making web apps, games, experiments, sounds. Hey there!

Game: Bleep Bloop

A clear-the-blocks style puzzle game based around matching and synchronizing rhythms. As the puzzle levels progress, the background chords and scale choices evolve, making the whole thing into a sort of interactive music piece. No game framework was used, just plain javascript - I pulled in a library called Tone.js to generate the synthesizer sounds in the browser, and the GreenSock animation library for all the animation and 'tweening. I am currently working on re-creating this in Unity for mobile.

Experiment: Multi-State Cellular Automata

Cellular automata are a way to model natural processes (among other things) by defining discrete "cells", in this case a grid, and setting up simple sets of rules that affect the states of each cell over time. The well-known example is John Conway's Game of Life, which involves only 2 cell states (alive or dead) and a few transition rules. This was my attempt to extend that into something with more cell states (colors) and more interesting ways of setting up rules.

Algorithm: Gale-Shapley Matching

An implementation of a well known solution to the Stable Marriage Problem, in which one group of elements is matched to a second group of elements by some criteria of preference, and must end up in a "stable" configuration of pairs such that there are no two elements that would prefer each other over their current match.

Generative Art: 10 Print

The famous line of BASIC code 10 PRINT CHR$(205.5+RND(1)); : GOTO 10 produces a random mazelike pattern by printing slashes and backslashes. It's so famous, someone wrote a book about it. This is my version of it.

Experiment: Black and White Shader Patterns

Each "channel" features a different set of computations carried out for each "pixel" to decide how light or dark it is. It's essentially a set of simple, black-and-white custom shaders.

Animation: Game Boy Music Video

A music video for a group I used to be in called Lowlands (check them out here). I did all the drawing in the style of a gameboy game, and painstakingly animated every frame by hand, a questionable decision indeed. My commitment to the genre homage led to replacing the actual song with a (also painstakingly hand-finagled) 16-bit version, which in retrospect seems a bit absurd as well. Nonetheless, it was fun to make and remains a relic of a simpler, more creative time in my life.
